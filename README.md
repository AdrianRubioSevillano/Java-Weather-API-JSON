# App del Clima (Java + API REST + JSON)

Este proyecto me ayudo a entender como conectar una aplicación Java con servicios externos de internet y trabajar con datos en tiempo real.

## Tecnologías aplicadas
* **Java:** Lenguaje principal.
* **Peticiones HTTP:** Para conectarme a la API externa y pedirle la información.
* **Procesamiento JSON:** Para traducir todo el bloque de texto que me devolvía la API y convertirlo en variables que Java pudiera entender.

## Funcionalidades mas interesantes

* **Manejar las APIs REST:** Crear la conexión, pedir los datos y procesar la respuesta.
* **Manejar los JSON:** Entendiendo los ficheros JSON diferenciando objetos, listas, etc.

## Como ejecutar el proyecto
Si te descargas el proyecto, el programa no funcionará directamente por motivos de seguridad. Para usarlo:
1. Créate una cuenta gratuita en [OpenWeatherMap](https://openweathermap.org/).
2. Genera tu propia clave.
3. Abre el archivo `Main.java` y, donde he dejado el comentario *"En este parametro se deberia introducir una API Key de OpenWeatherMap"*, pega tu clave real.
